## Import

```jsx
// or

```

## Props

| Name | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| id | `GridRowId` | - | Yes |  |
| indexRelativeToCurrentPage | `number` | - | Yes |  |
| isFirstVisible | `boolean` | - | Yes |  |
| isLastVisible | `boolean` | - | Yes |  |
| model | `R` | - | Yes |  |

> **Note**: The `ref` is forwarded to the root element.
